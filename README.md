# Client Lead Management System (Mini CRM)

## 📌 Internship Task
This project is part of the Future Interns Full Stack Web Development Internship (Task 2).

## 🌐 Project Overview
A simple CRM (Customer Relationship Management) system used to manage client leads, track status, and store basic client information.

## 🚀 Features
- Add new client leads
- View list of leads
- Update lead status (New / Contacted / Converted)
- Add notes for follow-ups
- Simple admin-style interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Flask / Node.js (if used)
- Database (SQLite / MongoDB)

## 🎯 Purpose
To understand CRUD operations, backend logic, and basic database handling in real-world applications.
